# Video Chat

This is a test project for learning how to create a Video chat website using django. Features include:

* Full user functionality  
  * Sign-up  
    * Email  
    * First Name  
    * Last Name  
    * Password  
  * Log-in  
  * Profile Page  
    * Able to edit profile   
* Create Rooms  
* Join Rooms  
  * Via link invite  
* Delete Rooms  
  * Room automatically delete after 24 Hours of inactivity
